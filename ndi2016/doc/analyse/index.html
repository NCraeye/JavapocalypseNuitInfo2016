Cahier des charges du site produit par l'équipe Javapocalypse lors de la nuit de l'info 2016

Besoins fonctionnels :

  • Pour le défi WebGL :

    • Accéder à un jeu de type infite runner pour un visiteur lamba.

  • Pour le défi Scrum :
    • Accéder à la répartition des sprints
    • Accéder à la répartition des actions faisables par un visiteur lambda : liste des user stories
    • Accéder à la répartition des tâches des membres de l'équipes
  • Pour le défi Erreur 404 :
    • Accéder à un site réactif, qui répond surtout quand l'url saisie est mauvaise.
  • Pour le défi Gamification :
    • Accéder à un site ludique, sur lequel on navigue de façon simple.

    Besoins non-fonctionnels :

    • Site responsive
    • Site simple d'utilisation
    • Outils de développement
      • Dépôt collaboratif : Git

        • Pour travailler en temps réel sur des documents partagés à 7 équipiers.

      • Côté serveur : framework Slim utilisant php

      • Côté client : template BootStrap + html/css
